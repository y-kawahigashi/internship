## 1. 会社概要

当社は、IT・技術系カンファレンスやセミナーの企画・運営を行うイベント運営会社です。年間を通じて、プログラミング言語のカンファレンス、クラウド技術セミナー、AI・機械学習ワークショップ、セキュリティ勉強会など、多様な技術イベントを開催しています。

主な開催イベント例：

- 技術カンファレンス（JavaScript Conf、Go Conference など）
- クラウドサービス関連セミナー（AWS、GCP、Azure）
- AI・機械学習ハンズオンワークショップ
- セキュリティ勉強会・CTF イベント
- OSS コミュニティミートアップ

## 2. プロジェクトの背景

当社では年間 50 件以上の技術イベントを開催していますが、現在イベント情報の発信は SNS、技術ブログ、メールマガジンなど複数のチャネルで行っており、参加希望者であるエンジニアの皆様にとって以下のような問題が発生しています。

- **イベント情報の分散**: 当社が開催するイベント情報が複数のプラットフォームに分散しており、興味のあるイベントを見つけにくい
- **情報へのアクセスの不便さ**: イベント情報を確認するために、SNS や各種サイトを巡回する必要があり、手間がかかる
- **目的のイベントを見つけにくい**: イベント数が多いため、自分が興味のある技術分野やスケジュールに合ったイベントを探すのに時間がかかる

これらの課題を解決するため、当社主催イベントの一覧を確認できる Web システムの導入を希望しています。

## 3. プロジェクトの目的

本プロジェクトでは、エンジニアの皆様が Web 上で当社主催の技術イベント情報を簡単に確認できる「イベント一覧表示システム」の開発を依頼します。

システムの主な目的は以下の通りです。

- **イベント情報の一元化**: 当社が開催するすべての技術イベント情報を一箇所に集約し、エンジニアの皆様が効率的にイベントを探せるようにする
- **イベント検索の効率化**: 検索・ソート機能により、エンジニアの皆様が自分の興味や予定に合ったイベントを素早く見つけられるようにする
- **ユーザー体験の向上**: 直感的な UI/UX により、エンジニアの皆様がスムーズにイベントを探せる環境を提供する

## 4. 解決したい課題

本システムの導入により、以下の課題を解決したいと考えています。

### 4.1 イベント情報の検索性向上

- **課題**: 当社のイベント情報が複数のプラットフォームに分散しており、エンジニアの皆様が興味のあるイベントを見つけにくい
- **解決策**: すべてのイベント情報を一箇所に集約し、エンジニアの皆様が効率的にイベントを探せるようにする

### 4.2 興味のあるイベントの発見しやすさ

- **課題**: 年間 50 件以上のイベントを開催しているため、エンジニアの皆様が自分の興味のある技術分野やスケジュールに合ったイベントを見つけるのに時間がかかる
- **解決策**: 検索・絞り込み・ソートなどの機能を提供することで、目的のイベントを素早く発見できるようにする

### 4.3 様々なデバイスからの利用

- **課題**: 業務中の PC からのアクセスだけでなく、移動中のスマートフォンからのアクセスも想定される
- **解決策**: 様々な画面サイズやデバイスに対応したレスポンシブ UI を提供することで、どのデバイスからでも快適にイベント情報を確認できるようにする

## 5. イベント情報として保持すべき内容

システムで管理するイベント情報として、以下の項目を保持してください。

- **イベント名**: カンファレンス名、セミナー名など
- **イベントの詳細な説明**: 対象技術、セッション内容、対象者レベルなど
- **イベント開始・終了日時**: 開催日時の情報
- **定員数**: イベントの参加可能人数

これらの情報により、エンジニアの皆様がイベントの内容や開催日時、定員状況を確認できるようにしてください。
