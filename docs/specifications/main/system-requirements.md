## 1. 概要

本ドキュメントは、インターンシップ実装課題「イベント一覧表示システム」の機能要件を定義します。

## 2. 目的・背景

本システムは、IT・技術系カンファレンスやセミナーへの参加を希望するエンジニアが、開催されるイベントを閲覧・検索するためのシステムです。
エンジニアは、本システムを通じて興味のある技術イベントを効率的に探すことができます。

## 3. 想定ユーザー

- IT・技術系イベントへの参加を希望するエンジニア

## 4. 業務フロー

1. ユーザーがイベント一覧画面にアクセスする
2. イベント一覧を閲覧する
3. 必要に応じて検索・ソート機能を使用し、興味のあるイベントを探す

詳細なワークフローについては、[業務フロー図](./workflow.md)を参照してください。

## 5. 機能要件

### 5.1 機能要件一覧

| 機能 ID | 機能グループ     | 機能名                 | 機能要件                                                                                                                                 |
| ------- | ---------------- | ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| 1.1     | イベント一覧表示 | イベント一覧取得       | 登録されているイベントの一覧を取得・確認できること                                                                                       |
| 1.2     |                  | イベント情報表示       | イベント情報として以下を確認できること<br>・イベント名<br>・イベント説明（存在する場合）<br>・開催開始日時<br>・開催終了日時<br>・定員数 |
| 1.3     |                  | イベント検索・絞り込み |                                                                                                                                          |
| 1.4     |                  | イベント一覧ソート     |                                                                                                                                          |

### 5.2 イベント情報の表示項目

| 項目名       | 必須 | 説明                             |
| ------------ | ---- | -------------------------------- |
| イベント名   | 必須 | カンファレンス名、セミナー名など |
| イベント説明 | 任意 | 対象技術、セッション内容など     |
| 開催開始日時 | 必須 | イベントの開催開始日時           |
| 開催終了日時 | 必須 | イベントの開催終了日時           |
| 定員数       | 必須 | イベントの参加可能人数           |

## 6. 非機能要件

本課題では取り扱わないため省略

## 7. 制約事項

- イベントデータは既にデータベースに登録されているものとする
